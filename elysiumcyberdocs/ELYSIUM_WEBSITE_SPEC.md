# Elysium Cyber Website - Requirements Specification

## Overview
Elysium Cyber (elysiumcyber.com) is a professional cybersecurity consulting firm website. The primary focus is rebuilding the /internship page as a professional recruitment page.

## Design Philosophy
- **Professional Fortune 500 style** - Like careers pages from major tech companies
- **NO pricing information** - This is not a paid product
- **NO "this isn't a course" language** - Unprofessional and defensive
- **NO salesy language** - Professional tone only
- **Clean, minimal, trustworthy** - Build credibility through professionalism

### Target Sections for /internship Page

#### 1. Hero Section
- Professional headline about the internship program
- Brief value proposition (1-2 sentences)
- Primary CTA: "Apply Now" or "Book a Call"
- Professional hero image

#### 2. Program Overview
- What the internship program is
- Duration and commitment
- What interns receive (mentorship, projects, reference letter)
- NO pricing or course language

#### 3. What Interns Work On
- Real project examples:
  - Cloud Security (AWS, S3, IAM)
  - Security Scripting (Python, Bash automation)
  - Compliance & Risk (NIST, CIS, SOC 2)
- Format: Cards or grid layout

#### 4. Who Should Apply
- Target candidates:
  - Students pursuing cybersecurity
  - Career changers entering security
  - IT professionals wanting security experience
- Clear requirements/qualifications

#### 5. Application Section
- Embedded application form OR
- Calendly integration for booking discovery calls
- Simple CTA to get started

#### 6. FAQ Section
- Common questions about the program
- Professional, informative answers
- Collapsible accordion format

## Brand Guidelines

### Tone
- Professional and confident
- Clear and direct
- No defensive language
- No sales pressure

### Visual Style
- Clean and minimal
- Plenty of whitespace
- Professional typography (Inter font)
- Consistent color scheme

### Avoid
- "This isn't a course"
- "Only $X"
- "Limited spots"
- "Act now"
- Excessive exclamation marks
- Emoji overuse
